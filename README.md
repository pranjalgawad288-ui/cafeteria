☕ Smart Cafeteria Ordering System

A modern QR Code-Based Smart Cafeteria Ordering System built with React and Supabase. The application allows customers to scan a QR code placed on their table, browse the digital menu, place food orders, and enables cafeteria staff to manage orders through an admin dashboard.

📖 Overview

The Smart Cafeteria Ordering System digitizes the food ordering process by eliminating paper menus and reducing waiting time. Customers simply scan a QR code to access the menu, add items to their cart, and place orders directly from their mobile devices.

The project demonstrates modern full-stack web development using React for the frontend and Supabase for backend services.
